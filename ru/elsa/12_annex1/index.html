    Приложение 1: Глоссарий¶

    Зоны ограниченных действий¶

    Зона землепользования, эквивалентная природоохранной мере, которая служит для улучшения конкретных объектов планирования. Зоны определяются ограничениями, которые устанавливают, где мера может или не может быть принята. Например, эти жесткие ограничения ограничивают охрану нетронутых территорий (например, с низким уровнем антропогенного воздействия) и охрану/восстановление территорий, которые подвергаются умеренному воздействию человеческой деятельности, но не полностью контролируются человеком (например, с низким и средним уровнем антропогенного воздействия).

    Площадной бюджет¶

    Максимальная площадь (выраженная в процентах от общей площади страны), которая может быть выделена для реализации того или иного мероприятия.

    Коэффициент граничного штрафа (BPF)¶

    Решения подвергаются “штрафу” в зависимости от общей длины внешней границы или края зон. Применяя штраф к решениям с большой длиной границы, данный коэффициент BPF может использоваться для поощрения пространственной связанности или кластеризации зон пространственной приоритизации при определении приоритетных территорий для реализации Рамочной программы.

    Программное обеспечение для поддержки принятия решений¶

    Компьютерное приложение, которое использует информацию о возможных действиях и ограничениях этих действий для содействия процессу принятия решений по достижению заявленной цели.

    Географическая информационная система (ГИС)¶

    Компьютерная система, состоящая из аппаратного и программного обеспечения, необходимого для сбора, хранения, управления, анализа и представления географических (пространственных) данных.

    Варианты блокировки¶

    Конкретные характеристики или области, которые закреплены в решении. Обычно это относится к существующим охраняемым территориям, которые по умолчанию закреплены в создании проблемы SCP.

    Планировочный элемент¶

    Пространственный набор данных, используемый для отображения пространственных элементов целей 1-12 Рамочной программы. Каждая цель Рамочной программы может быть отображена с помощью одной или нескольких планируемых характеристик в зависимости от ее сложности. Планируемые характеристики могут включать экологические классификации, типы среды обитания, виды, физические объекты, процессы или любые элементы, которые можно измерить в единице планирования. 

    Планировочные единицы¶

    Планировочные единицы являются основными элементами анализа приоритетов. Исследуемая территория делится на планировочные единицы, которые представляют собой небольшие географические участки правильной или неправильной формы. Примерами могут служить квадраты, шестиугольники, кадастровые участки и гидрологические единицы.

    Представление¶

    В системном планировании природоохранных мероприятий репрезентативная система охватывает весь спектр планируемых объектов (видов, экосистем и экосистемных услуг), встречающихся в регионе планирования, а не только знаковые виды.

    Ограничения¶

    Ограничение, которое должно быть соблюдено в ходе оптимизации. Основные ограничения заключаются в том, что бюджет (площадь земли, выделенная для каждого действия) не должен быть превышен и что каждая зона может существовать только в пределах определенных планировочных единиц (например, охранная зона может быть создана только в планировочных единицах, которые не являются сельскохозяйственными или городскими территориями).

    Систематическое планирование природоохранных действий (SCP){#systematic-conservation-planning-scp}¶

    Формальный метод выявления потенциальных территорий для природоохранного управления, которые наиболее эффективно позволят достичь конкретного набора целей, обычно минимального представления биоразнообразия. Этот процесс предполагает четкий и структурированный подход к установлению приоритетов и в настоящее время является нормой как для охраны суши, так и для охраны морской среды. Эффективность систематического планирования охраны природы заключается в его способности максимально эффективно использовать ограниченные финансовые ресурсы для достижения целей охраны природы, причем таким образом, который является обоснованным, подотчетным и прозрачно учитывает потребности различных пользователей ресурсов.

    Пользовательский интерфейс¶

    Средство, с помощью которого пользователи взаимодействуют с определенным компьютерным приложением. Графический интерфейс пользователя (GUI) представляет информацию в простой форме с использованием графики, меню и значков.

    Веса¶

    Веса позволяют пользователям устанавливать относительные приоритеты для различных функций планирования. Значения обычно варьируются от «0» (не имеет значения) до «10» (имеет чрезвычайно высокое значение).
