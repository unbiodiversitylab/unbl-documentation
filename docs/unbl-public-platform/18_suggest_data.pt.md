# Como faço para sugerir mais dados para inclusão no Laboratório de Biodiversidade da ONU?

Recebemos com satisfação sugestões sobre dados que enriqueceriam o Laboratório de Biodiversidade da ONU! Se seus dados abordam biodiversidade, conservação ou bem-estar humano em níveis nacional, regional ou internacional, preencha [este formulário](https://forms.gle/AFqQf5dPZySQnhdq9).

Nossa equipe revisará as sugestões conforme forem recebidas. O PNUD e o UNEP-WCMC têm critérios formalizados para dados publicados na plataforma pública para auxiliar na identificação e curadoria dos melhores conjuntos de dados disponíveis sobre esses temas, a fim de permitir análises e relatórios simplificados e padronizados. Os critérios estão disponíveis [aqui](https://unbiodiversitylab.org/wp-content/uploads/2021/01/Global-Biodiversity-Data-Fitness-Assessment.pdf).

Para carregar dados para uso de sua instituição em um espaço de trabalho do UNBL, consulte a próxima pergunta sobre como solicitar um espaço de trabalho.
