Existem vários recursos que podem ajudá-lo a navegar pela tela do mapa. Estes incluem:

1. *Mover mapa*: Use o mouse para clicar e arrastar a parte do mapa que você gostaria de ver para o meio da tela.

2. *Aumentar/diminuir zoom:* Clique nos ícones +/- no canto inferior direito do mapa ou use a roda de rolagem do mouse.

3. *Centralizar local:* Clique no botão ![](images/icons/center-place-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} acima de +/-. Se você selecionou um local na barra de menu esquerda, isso centralizará novamente o mapa sobre o local selecionado.

4. *Ocultar barra de menu esquerda:* Clique na seta no topo do menu esquerdo para recolher o painel de conjuntos de dados para uma visualização maior do mapa. Clique novamente para expandir o painel.
