# ¿Cómo sugiero más datos para incluir en el Laboratorio de Biodiversidad de las Naciones Unidas?

¡Damos la bienvenida a sugerencias sobre datos que enriquecerían el Laboratorio de Biodiversidad de las Naciones Unidas! Si sus datos abordan la biodiversidad, la conservación o el bienestar humano a nivel nacional, regional o internacional, complete [este formulario](https://forms.gle/AFqQf5dPZySQnhdq9).

Nuestro equipo revisará las sugerencias a medida que se reciban. El PNUD y el PNUMA-WCMC han formalizado criterios para los datos publicados en la plataforma pública para ayudar a identificar y curar los mejores conjuntos de datos disponibles sobre estos temas para permitir análisis e informes simplificados y estandarizados. Los criterios están disponibles [aquí](https://unbiodiversitylab.org/wp-content/uploads/2021/01/Global-Biodiversity-Data-Fitness-Assessment.pdf).
