# Как найти дополнительные наборы данных для моей страны?

Данные в Лаборатории биоразнообразия ООН включают наилучшие доступные глобальные наборы данных, связанные с природой и благосостоянием человека, начиная от биоразнообразия до экосистемных услуг и социально-экономических данных. Мы также включаем региональные наборы данных, когда это рекомендовано пользователями Лаборатории биоразнообразия ООН. Вы можете просматривать наборы данных в Лаборатории биоразнообразия ООН глобально или в пределах интересующей области.

!!! Note
	В этом руководстве и в UNBL мы ссылаемся как на наборы данных, так и на слои данных. Каждый набор данных может содержать один или несколько слоев данных.

1.	Перейдите к вашей интересующей области, если предпочтительно. Вы также можете оставаться в глобальном виде.

2.	Нажмите на значок вкладки 'DATASETS'.

3.	Для поиска набора данных вы можете:

	a) Введите название набора данных, который вы хотите просмотреть, в поле поиска и выберите желаемый результат в списке наборов данных (примечание: ваш поиск должен включать не менее 3 символов).

    **ИЛИ**

    b) Нажмите для раскрытия фильтров, чтобы просмотреть и выбрать категории наборов данных, которые вас интересуют. Затем вы можете выбрать желаемый набор данных из списка результатов поиска.

	![](images/en/image043.png)

	**ИЛИ**

	c) Нажмите для раскрытия тегов наборов данных и выберите интересующий вас тег. Затем вы можете выбрать желаемый набор данных из списка.

	![](images/en/image045.png)
	![](images/en/image047.png)

4.	Нажмите кнопку переключения справа от названия набора данных, чтобы загрузить этот набор данных на карту.

5.	Нажмите кнопку переключения снова или нажмите значок **X** в легенде набора данных, чтобы удалить этот набор данных.

	![](images/en/image049.png)
