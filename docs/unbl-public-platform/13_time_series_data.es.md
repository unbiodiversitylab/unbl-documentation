# ¿Qué opciones tengo para visualizar conjuntos de datos de series temporales?

El Laboratorio de Biodiversidad de las Naciones Unidas le proporciona acceso a conjuntos de datos que muestran cambios a lo largo del tiempo. Algunos conjuntos de datos de series temporales se visualizan a lo largo de múltiples años con animación, otros se pueden visualizar por año específico a través del menú desplegable, y algunos son una combinación de ambos con la capacidad de visualizar animaciones de años particulares que se pueden elegir del menú desplegable.

1.  Seleccione la etiqueta "Series temporales" en la pestaña de filtros para filtrar los conjuntos de datos que están disponibles como series temporales.

2.	Seleccione el conjunto de datos de interés.
3.	Personalice según las opciones disponibles:

	a)	*Solo animación*: Haga clic en el icono de reproducción a la izquierda para ver la animación de cambios durante este período de tiempo. Seleccione un tiempo específico (año, mes o fecha) que desee mostrar en el mapa haciendo clic directamente en la barra de línea de tiempo. Para visualizar un rango de tiempo personalizado, seleccione un tiempo específico directamente en la barra de línea de tiempo y luego haga clic en el icono de reproducción para ver los cambios durante este período de tiempo.

	![](images/en/image069.png)

	**O**

	b)	*Menú desplegable*: Seleccione un año específico que desee mostrar en el mapa eligiéndolo de las capas disponibles en el menú desplegable en la leyenda. Solo se puede visualizar una capa de período de tiempo único usando esta opción.
