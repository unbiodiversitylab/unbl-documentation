# Comment ajuster ma vue cartographique ?

Plusieurs fonctionnalités peuvent vous aider à naviguer sur l'écran de carte. Celles-ci incluent :

1. *Déplacer la carte* : Utilisez votre souris pour cliquer et faire glisser la partie de la carte que vous souhaitez voir au milieu de l'écran.

2. *Zoomer/Dézoomer :* Cliquez sur les icônes +/- en bas à droite de la carte ou utilisez la molette de défilement de votre souris.

3. *Centrer le lieu :* Cliquez sur le bouton de centrage du lieu au-dessus de +/-. Si vous avez sélectionné un lieu dans la barre de menu de gauche, cela recentrera la carte sur le lieu sélectionné.

4. *Masquer la barre de menu de gauche :* Cliquez sur la flèche en haut du menu de gauche pour réduire le panneau de données et obtenir une vue cartographique plus large. Cliquez à nouveau pour développer le panneau.
