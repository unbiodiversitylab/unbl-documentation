# Quelles options ai-je pour visualiser les ensembles de données de séries chronologiques ?

Le Laboratoire de Biodiversité des Nations Unies vous donne accès à des ensembles de données qui montrent les changements au fil du temps. Certains ensembles de données de séries chronologiques sont visualisés sur plusieurs années avec animation, d'autres peuvent être visualisés par année spécifique via le menu déroulant, et certains sont une combinaison des deux avec la possibilité de visualiser des animations d'années particulières qui peuvent être choisies dans le menu déroulant.

1.  Sélectionnez la balise "Time series" dans l'onglet des filtres pour filtrer les ensembles de données disponibles sous forme de séries chronologiques.

2.	Sélectionnez l'ensemble de données d'intérêt.
3.	Personnalisez en fonction des options disponibles :

	a)	*Animation uniquement* : Cliquez sur l'icône de lecture à gauche pour voir l'animation des changements au cours de cette période. Sélectionnez un moment spécifique (année, mois ou date) que vous souhaitez afficher sur la carte en cliquant directement sur la barre de la chronologie. Pour visualiser une plage de temps personnalisée, sélectionnez un moment spécifique directement sur la barre de la chronologie, puis cliquez sur l'icône de lecture pour voir les changements au cours de cette période.

	![](images/en/image069.png)

	**OU**

	b)	*Menu déroulant* : Sélectionnez une année spécifique que vous souhaitez afficher sur la carte en la choisissant parmi les couches disponibles dans le menu déroulant de la légende. Une seule couche de période temporelle peut être visualisée en utilisant cette option.
