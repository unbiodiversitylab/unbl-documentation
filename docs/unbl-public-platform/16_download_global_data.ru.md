# Как загрузить необрезанные глобальные наборы данных?

1.	Выберите интересующий набор данных.

2.	Нажмите на значок информации о наборе данных.

3.	Нажмите на ссылку в разделе 'LEARN MORE' для загрузки данных из их исходного источника (если ссылка не предоставлена, то это, скорее всего, означает, что данные не являются общедоступными для загрузки, или поставщики данных отозвали разрешения на включение ссылки для загрузки в метаданные набора данных в Лаборатории биоразнообразия ООН).

4.	Если вы столкнулись с какими-либо проблемами при доступе к данным, пожалуйста, свяжитесь с support@unbiodiversitylab.org для дальнейшей поддержки.
