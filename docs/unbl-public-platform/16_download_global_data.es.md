# ¿Cómo descargo conjuntos de datos globales sin recortar?

1.	Seleccione el conjunto de datos de interés.

2.	Haga clic en el icono de información del conjunto de datos.

3.	Haga clic en el enlace debajo de 'MÁS INFORMACIÓN' para descargar los datos de su fuente original (si no se proporciona ningún enlace, esto probablemente significa que los datos no están disponibles públicamente para descargar o los proveedores de datos han renunciado a los permisos para incluir el enlace de descarga en los metadatos del conjunto de datos en el Laboratorio de Biodiversidad de las Naciones Unidas).

4.	Si encuentra algún problema para acceder a los datos, comuníquese con support@unbiodiversitylab.org para obtener más ayuda.
