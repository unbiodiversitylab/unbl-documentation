# Какие варианты визуализации временных рядов данных доступны?

Лаборатория биоразнообразия ООН предоставляет вам доступ к наборам данных, которые показывают изменения во времени. Некоторые наборы данных временных рядов визуализируются на протяжении нескольких лет с анимацией, другие могут быть визуализированы по конкретному году через выпадающее меню, а некоторые представляют собой комбинацию обоих с возможностью визуализации анимаций конкретных лет, которые можно выбрать из выпадающего меню.

1.  Выберите тег "Time series" на вкладке фильтров для фильтрации наборов данных, которые доступны как временные ряды.

2.	Выберите интересующий набор данных.
3.	Настройте на основе доступных опций:

	a)	*Только анимация*: Нажмите на значок воспроизведения слева, чтобы увидеть анимацию изменений за этот период времени. Выберите конкретное время (год, месяц или дату), которое вы хотите показать на карте, нажав непосредственно на временной шкале. Чтобы визуализировать настраиваемый временной диапазон, выберите конкретное время непосредственно на временной шкале, а затем нажмите на значок воспроизведения, чтобы увидеть изменения за этот период времени.

	![](images/en/image069.png)

	**ИЛИ**

	b)	*Выпадающее меню*: Выберите конкретный год, который вы хотите показать на карте, выбрав его из доступных слоев в выпадающем меню в легенде. Только один слой временного периода может быть визуализирован с использованием этой опции.
