# Как найти дополнительную информацию о каждом наборе данных?

1.	Выберите набор данных и загрузите его на карту.

2.	В нижнем левом углу вида карты будет легенда, показывающая название и символику всех наборов данных, которые в настоящее время включены на карте.  Нажмите на значок ![](images/icons/info-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} для просмотра информации о наборе данных. Также вы можете нажать на тот же значок ![](images/icons/info-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} непосредственно рядом с кнопкой переключения для каждого набора данных на вкладке поиска наборов данных. Информация предоставляет описание набора данных, организацию-источник, цитирования и ссылки для загрузки данных.
