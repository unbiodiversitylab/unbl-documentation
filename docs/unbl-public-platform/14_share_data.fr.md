# Comment partager un ensemble de données ?

Le partage d'ensembles de données disponibles sur la plateforme publique est facile. Pour ce faire :

1.	Activez le(s) ensemble(s) de données d'intérêt dans votre vue cartographique.

2.	Copiez l'URL affichée dans votre navigateur.

3.	Partagez ! Tous les ensembles de données sur notre plateforme publique seront accessibles par n'importe qui, qu'il soit ou non un utilisateur enregistré.
