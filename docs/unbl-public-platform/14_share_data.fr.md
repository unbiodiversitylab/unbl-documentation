Le partage des ensembles de données disponibles sur la plateforme publique est facile. Pour ce faire :

1. Activez le ou les ensembles de données qui vous intéressent dans votre visualiseur de carte.

2. Copiez l'URL affichée dans votre navigateur.

3. Partagez ! Tous les ensembles de données de notre plateforme publique sont accessibles à tous, que vous soyez ou non un utilisateur enregistré.
