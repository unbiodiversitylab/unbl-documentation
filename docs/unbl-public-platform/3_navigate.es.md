# ¿Cómo navego entre el sitio web del Laboratorio de Biodiversidad de las Naciones Unidas y la aplicación de mapas?

Navegar entre las dos páginas es simple:

1. Para volver al sitio web del Laboratorio de Biodiversidad de las Naciones Unidas desde la aplicación de mapas, haga clic en 'VISTA DE MAPA' en la barra de herramientas del lado izquierdo y elija 'INICIO' en la parte superior derecha del panel.

	!!! Note
		Si está registrado en UNBL y posee un espacio de trabajo, haga clic en 'ESPACIOS DE TRABAJO' en la barra de herramientas del lado izquierdo y luego en 'INICIO'.

	![](images/en/image023.png)

2. Para navegar a la aplicación de mapas desde el sitio web del Laboratorio de Biodiversidad de las Naciones Unidas, haga clic en 'Iniciar mapa'.

	![](images/en/image025.png)
