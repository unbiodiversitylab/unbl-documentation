1.	Selecione o conjunto de dados e carregue-o no mapa.

2.	No canto inferior esquerdo da visualização do mapa, haverá uma legenda mostrando o nome e a simbologia de todos os conjuntos de dados atualmente ativados no mapa. Clique no ícone ![](images/icons/info-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} para visualizar as informações do conjunto de dados. Alternativamente, clique no mesmo ícone ![](images/icons/info-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} diretamente ao lado do botão de alternância para cada conjunto de dados na aba de pesquisa de conjuntos de dados. As informações fornecem uma descrição do conjunto de dados, organização de origem, citações e links para baixar os dados.
