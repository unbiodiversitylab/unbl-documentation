Compartilhar conjuntos de dados que estão disponíveis na plataforma pública é fácil. Para fazer isso:

1.	Ative o(s) conjunto(s) de dados de interesse na visualização do seu mapa.

2.	Copie a URL exibida no seu navegador.

3.	Compartilhe! Quaisquer conjuntos de dados em nossa plataforma pública estarão acessíveis a qualquer pessoa, independentemente de serem um usuário registrado.
