# Как мне предложить дополнительные данные для включения в Лабораторию ООН по биоразнообразию?

Мы приветствуем предложения по данным, которые обогатят Лабораторию ООН по биоразнообразию! Если ваши данные касаются биоразнообразия, охраны природы или благосостояния людей на национальном, региональном или международном уровне, пожалуйста, заполните [эту форму](https://forms.gle/AFqQf5dPZySQnhdq9).

Наша команда рассмотрит предложения по мере их поступления. ПРООН и ЮНЕП-WCMC формализовали критерии для данных, публикуемых на публичной платформе, чтобы помочь в выявлении и отборе лучших доступных наборов данных по этим темам для обеспечения оптимизированного и стандартизированного анализа и отчетности. Критерии доступны [здесь](https://unbiodiversitylab.org/wp-content/uploads/2021/01/Global-Biodiversity-Data-Fitness-Assessment.pdf).
