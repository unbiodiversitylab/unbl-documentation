# Как предложить больше данных для включения в Лабораторию биоразнообразия ООН?

Мы приветствуем предложения о данных, которые обогатят Лабораторию биоразнообразия ООН! Если ваши данные касаются биоразнообразия, сохранения или благосостояния человека на национальном, региональном или международном уровнях, пожалуйста, заполните [эту форму](https://forms.gle/AFqQf5dPZySQnhdq9).

Наша команда будет рассматривать предложения по мере их поступления. ПРООН и UNEP-WCMC имеют формализованные критерии для данных, публикуемых на публичной платформе, чтобы помочь в выявлении и курировании лучших доступных наборов данных по этим темам для обеспечения упорядоченного и стандартизированного анализа и отчетности. Критерии доступны [здесь](https://unbiodiversitylab.org/wp-content/uploads/2021/01/Global-Biodiversity-Data-Fitness-Assessment.pdf).

Чтобы загрузить данные для использования вашим учреждением в рабочем пространстве UNBL, пожалуйста, см. следующий вопрос о запросе рабочего пространства.
