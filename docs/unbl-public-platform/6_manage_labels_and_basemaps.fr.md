# Comment ajouter/supprimer les étiquettes de lieux, les routes et la vue satellite du fond de carte ?

Il existe plusieurs options pour personnaliser le fond de carte. Celles-ci sont disponibles sous l'icône 'MAP CONTROLS' en bas à droite et incluent :

1. *Étiquettes* : Les étiquettes affichent le nom des lieux, y compris les pays, les états, les villes et les points de repère représentatifs. Cliquez sur le bouton à bascule pour activer les étiquettes, et désactivez pour les masquer.

2. *Routes* : Cliquez sur le bouton à bascule pour afficher les routes ; désactivez pour masquer les routes.

3. *Fond de carte* : Nous proposons des options de fond de carte en niveaux de gris et en satellite. Cliquez sur le bouton à bascule pour activer le fond de votre choix.
