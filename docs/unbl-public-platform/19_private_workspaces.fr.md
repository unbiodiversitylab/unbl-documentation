# Que sont les espaces de travail UNBL ? Comment demander un espace de travail ?

Les espaces de travail UNBL fournissent une zone de travail sécurisée où des données de toute échelle peuvent être ajoutées et partagées avec un ensemble d'utilisateurs spécifiés. Ils offrent une zone sécurisée pour collaborer sur vos projets et peuvent fournir un moyen pour les experts SIG de collaborer avec des experts non techniques en utilisant une interface conviviale.

**Trois fonctionnalités sont disponibles via les espaces de travail qui ne sont pas disponibles sur la plateforme publique** :

1. Possibilité de télécharger des lieux ;

2. Possibilité d'ajouter vos propres ensembles de données et de les partager de manière privée et/ou publique ;

3. Possibilité d'ajouter notre outil de planification spatiale intégrée ELSA pour exécuter une analyse de priorisation spatiale pour votre pays.

Les espaces de travail UNBL sont disponibles pour tous les utilisateurs non commerciaux. Pour demander un espace de travail, veuillez naviguer vers la [page des espaces de travail du Laboratoire de Biodiversité des Nations Unies](https://unbiodiversitylab.org/en/unbl-workspaces/) et remplir le formulaire.

Vous pouvez accéder à notre guide d'utilisation sur les espaces de travail UNBL [ici](../unbl-workspaces/index.md).
