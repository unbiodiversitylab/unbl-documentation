# Comment changer la langue ?

Le Laboratoire de Biodiversité des Nations Unies est actuellement disponible en **anglais**, **français**, **espagnol**, **portugais** et **russe**. La langue par défaut est l'**anglais**.

Pour changer la langue, cliquez sur l'icône de compte dans le coin droit de la carte et cliquez à nouveau pour sélectionner la langue de votre choix dans le menu déroulant. Vous pouvez changer votre langue soit sur le site web du Laboratoire de Biodiversité des Nations Unies, soit sur l'application cartographique.
