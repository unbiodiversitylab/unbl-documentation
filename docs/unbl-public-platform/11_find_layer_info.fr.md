# Comment trouver plus d'informations sur chaque ensemble de données ?

1.	Sélectionnez l'ensemble de données et chargez-le sur la carte.

2.	Dans le coin inférieur gauche de la vue cartographique, il y aura une légende montrant le nom et la symbologie de tous les ensembles de données actuellement activés sur la carte. Cliquez sur l'icône ![](images/icons/info-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} pour afficher les informations sur l'ensemble de données. Alternativement, cliquez sur la même icône ![](images/icons/info-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} directement à côté du bouton à bascule pour chaque ensemble de données dans l'onglet de recherche des ensembles de données. Les informations fournissent une description de l'ensemble de données, l'organisation source, les citations et les liens pour télécharger les données.
