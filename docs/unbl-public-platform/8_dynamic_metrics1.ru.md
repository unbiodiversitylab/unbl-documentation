# Какие динамические показатели доступны для моей страны/интересующей области?

UNBL предлагает наглядные показатели на основе наилучших доступных глобальных пространственных наборов данных. Эти показатели могут использоваться для отчетности о состоянии природы и человеческого развития для мест, доступных на публичной платформе UNBL, и/или тех, которые вы загрузили в свое рабочее пространство (см. наше [руководство по рабочим пространствам](../unbl-workspaces/index.md) для получения дополнительной информации об этом). Доступные стандартные показатели включают:

- Глобальный земельный покров (2022)
- Изменение земельного покрова (1992-2022)
- Охраняемые территории (2025)
- Потеря древесного покрова (2001-2024)
- Ежемесячная активность пожаров (2023)
- Индекс биоразнообразия в естественном состоянии (2015)
- Плотность наземного углерода (2010)
- Улучшенный индекс растительности (2001-2022)
- Наземный индекс человеческого промышленного воздействия (2000, 2013)

Лаборатория биоразнообразия ООН дополнительно предлагает два основных индикатора, которые доступны в соответствии с метаданными индикаторов, связанных с Системой мониторинга Куньмин-Монреальской глобальной рамочной программы в области биоразнообразия ([CBD/DEC/COP/15/5](https://www.cbd.int/doc/decisions/cop-15/cop-15-dec-05-en.pdf); [CBD/DEC/COP/16/31](https://www.cbd.int/doc/decisions/cop-16/cop-16-dec-31-en.pdf)), которая доступна на [веб-сайте индикаторов Куньмин-Монреальской глобальной рамочной программы в области биоразнообразия](https://gbf-indicators.org/) и в [CBD/COP/16/INF/3/Rev.1](https://www.cbd.int/doc/c/ea34/8414/8c5e6797d291af15f33d6e40/cop-16-inf-03-rev1-en.pdf):

- Устойчивое сельское хозяйство (Основной индикатор 10.1)
- Устойчивое лесопользование (Основной индикатор 10.2)

Важно отметить, что восемь стандартных показателей могут отображаться для мест любого типа (стран, административных областей любого масштаба, географических областей и т.д.), в то время как два основных индикатора и показатель охраняемых территорий могут отображаться только для мест на уровне страны. Чтобы узнать больше о наборах данных, лежащих в основе каждого из этих показателей, и о том, как показатели могут использоваться для мониторинга и отчетности, пожалуйста, см. таблицу ниже.

Чтобы просмотреть эти показатели в Лаборатории биоразнообразия ООН:

1.	Выберите конкретную страну или интересующую область на вкладке 'PLACES'.

2.	Просмотрите показатели на левой панели. Выберите между списком девяти динамических показателей или двух основных индикаторов, нажав на кнопку 'METRICS' или 'HEADLINE INDICATORS'. Обратите внимание, что основные индикаторы и стандартный показатель охраняемых территорий могут отображаться только для мест типа страны.

3.	Нажмите на кнопку переключения рядом с любым конкретным показателем, если вы хотите просмотреть этот набор данных на карте. Нажмите на кнопку переключения снова или значок удаления набора данных в легенде, чтобы очистить экран.

	![](images/en/image035.png)

4.  Нажмите на значок ![](images/icons/info-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} для просмотра информации о наборе данных. Информационные страницы предоставляют краткое описание данных, связанную статью для чтения, необработанные данные для загрузки (если свободно доступны) и спецификации лицензии.

5.	Чтобы загрузить сводные данные для показателя в формате .csv или .json, нажмите на значок стрелки ![](images/icons/download-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"}. Вы также можете загрузить данные по исходным ссылкам на информационных страницах наборов данных.

	![](images/en/image041.png)
