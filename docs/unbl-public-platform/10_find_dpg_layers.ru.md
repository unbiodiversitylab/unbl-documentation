# Как найти открытые наборы данных Цифровых общественных благ (DPG)?

Лаборатория биоразнообразия ООН сертифицирована как платформа Цифровых общественных благ (DPG) Альянсом цифровых общественных благ. Она предоставляет новые способы для политиков взаимодействовать с высококачественными пространственными данными. Вы можете просматривать пространственные наборы данных, признанные открытыми данными DPG, в глобальном масштабе или в пределах интересующей области.

1.	Нажмите на значок вкладки 'DATASETS'.

2.	Нажмите для раскрытия фильтров.

3.	В фильтрах нажмите для раскрытия тегов наборов данных. Затем выберите "DPG Open Data".

	![](images/en/image051.png)

4.	Сверните список 'FILTERS' и просмотрите список открытых данных DPG в UNBL.

5.	Нажмите кнопку переключения справа от названия набора данных, чтобы загрузить интересующий набор данных на карту.

6.	Нажмите кнопку переключения снова или нажмите значок **X** в легенде набора данных, чтобы удалить этот набор данных.

	![](images/en/image053.png)
