# Como faço para baixar conjuntos de dados globais não recortados?

1.	Selecione o conjunto de dados de interesse.

2.	Clique no ícone de informações do conjunto de dados.

3.	Clique no link em 'SAIBA MAIS' para baixar os dados de sua fonte original (se nenhum link for fornecido, isso provavelmente significa que os dados não estão disponíveis publicamente para download ou os provedores de dados renunciaram às permissões para incluir o link de download nos metadados do conjunto de dados no Laboratório de Biodiversidade da ONU).

4.	 Se você encontrar quaisquer problemas ao acessar os dados, entre em contato com support@unbiodiversitylab.org para obter suporte adicional.
