# Comment trouver les couches de données ouvertes des Biens Publics Numériques (DPG) ?

Le Laboratoire de Biodiversité des Nations Unies est certifié comme plateforme de Biens Publics Numériques (DPG) par l'Alliance des Biens Publics Numériques. Il offre de nouvelles façons pour les décideurs politiques d'interagir avec des données spatiales de haute qualité. Vous pouvez visualiser les ensembles de données spatiales reconnus comme données ouvertes DPG à l'échelle mondiale ou dans une zone d'intérêt.

1.	Cliquez sur l'icône de l'onglet 'DATASETS'.

2.	Cliquez pour développer les filtres.

3.	Dans les filtres, cliquez pour développer les balises d'ensembles de données. Ensuite, sélectionnez "DPG Open Data".

	![](images/en/image051.png)

4.	Réduisez la liste 'FILTERS' et visualisez la liste des données ouvertes DPG sur UNBL.

5.	Cliquez sur le bouton à bascule à droite du nom de l'ensemble de données pour charger l'ensemble de données d'intérêt sur la carte.

6.	Cliquez à nouveau sur le bouton à bascule ou cliquez sur l'icône **X** dans la légende de l'ensemble de données pour supprimer cet ensemble de données.

	![](images/en/image053.png)
