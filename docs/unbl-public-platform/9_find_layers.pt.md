# Como faço para encontrar conjuntos de dados adicionais para meu país?

Os dados no Laboratório de Biodiversidade da ONU incluem os melhores conjuntos de dados globais disponíveis relacionados à natureza e ao bem-estar humano, variando de biodiversidade a serviços ecossistêmicos, a dados socioeconômicos. Também incluímos conjuntos de dados regionais quando recomendado pelos usuários do Laboratório de Biodiversidade da ONU. Você pode visualizar conjuntos de dados no Laboratório de Biodiversidade da ONU globalmente ou dentro de uma área de interesse.

!!! Note
	Referimo-nos tanto a conjuntos de dados quanto a camadas de dados neste guia e no UNBL. Cada conjunto de dados pode ter uma ou múltiplas camadas de dados dentro dele.

1.	Navegue até sua área de interesse, se preferir. Você também pode permanecer na visualização global.

2.	Clique no ícone da aba 'CONJUNTOS DE DADOS'.

3.	Para pesquisar um conjunto de dados, você pode:

	a) Digite o nome do conjunto de dados que você deseja visualizar na caixa de pesquisa e selecione o resultado desejado na lista de conjuntos de dados (nota: sua pesquisa deve incluir pelo menos 3 caracteres).

    **OU**

    b) Clique para expandir os filtros para visualizar e selecionar as categorias de conjuntos de dados de seu interesse. Em seguida, você pode selecionar o conjunto de dados desejado na lista de resultados da pesquisa.

	![](images/en/image043.png)

	**OU**

	c) Clique para expandir as tags de conjuntos de dados e selecione sua tag de interesse. Em seguida, você pode selecionar o conjunto de dados desejado na lista.

	![](images/en/image045.png)
	![](images/en/image047.png)

4.	Clique no botão de alternância à direita do nome do conjunto de dados para carregar este conjunto de dados no mapa.

5.	Clique no botão de alternância novamente ou clique no ícone **X** na legenda do conjunto de dados para remover este conjunto de dados.

	![](images/en/image049.png)
