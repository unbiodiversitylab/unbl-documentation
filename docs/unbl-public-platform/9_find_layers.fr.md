# Comment trouver des ensembles de données supplémentaires pour mon pays ?

Les données sur le Laboratoire de Biodiversité des Nations Unies incluent les meilleurs ensembles de données mondiales disponibles liés à la nature et au bien-être humain, allant de la biodiversité aux services écosystémiques, en passant par les données socio-économiques. Nous incluons également des ensembles de données régionales lorsque recommandés par les utilisateurs du Laboratoire de Biodiversité des Nations Unies. Vous pouvez visualiser les ensembles de données sur le Laboratoire de Biodiversité des Nations Unies à l'échelle mondiale ou dans une zone d'intérêt.

!!! Note
	Nous faisons référence à la fois aux ensembles de données et aux couches de données dans ce guide et sur UNBL. Chaque ensemble de données peut avoir une ou plusieurs couches de données.

1.	Naviguez vers votre zone d'intérêt, si vous le souhaitez. Vous pouvez également rester sur la vue mondiale.

2.	Cliquez sur l'icône de l'onglet 'DATASETS'.

3.	Pour rechercher un ensemble de données, vous pouvez soit :

	a) Taper le nom de l'ensemble de données que vous souhaitez visualiser dans la zone de recherche et sélectionner le résultat souhaité dans la liste des ensembles de données (remarque : votre recherche doit contenir au moins 3 caractères).

    **OU**

    b) Cliquer pour développer les filtres pour visualiser et sélectionner les catégories d'ensembles de données de votre intérêt. Vous pouvez ensuite sélectionner l'ensemble de données souhaité dans la liste des résultats de recherche.

	![](images/en/image043.png)

	**OU**

	c) Cliquer pour développer les balises d'ensembles de données et sélectionner votre balise d'intérêt. Vous pouvez ensuite sélectionner l'ensemble de données souhaité dans la liste.

	![](images/en/image045.png)
	![](images/en/image047.png)

4.	Cliquez sur le bouton à bascule à droite du nom de l'ensemble de données pour charger cet ensemble de données sur la carte.

5.	Cliquez à nouveau sur le bouton à bascule ou cliquez sur l'icône **X** dans la légende de l'ensemble de données pour supprimer cet ensemble de données.

	![](images/en/image049.png)
