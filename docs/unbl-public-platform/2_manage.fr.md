# Comment gérer mon compte ?

Une fois inscrit au Laboratoire de Biodiversité des Nations Unies, vous pourrez gérer votre compte, y compris modifier votre nom d'utilisateur, email, mot de passe, pays et institution. Vous pourrez également consulter et modifier les espaces de travail auxquels vous appartenez.

**Pour gérer votre compte :**

1. Cliquez sur l'icône de compte avec vos initiales en haut à droite, puis cliquez sur 'Profile'.

	![](images/en/image017.png)

2. Cliquez sur 'EDIT' pour modifier votre nom d'utilisateur, email, pays et institution.

3. Pour réinitialiser votre mot de passe, cliquez sur 'SEND RESET EMAIL', puis suivez les instructions dans l'email.

4. Pour quitter l'un des espaces de travail UNBL auxquels vous appartenez, cliquez sur 'EDIT', puis sur 'LEAVE WORKSPACE'. Enregistrez vos modifications.

	![](images/en/image021.png)

5. Si ce compte n'est plus utilisé, vous pouvez cliquer sur 'DELETE YOUR ACCOUNT' en bas de cette page. Après avoir supprimé le compte, vous devrez vous réinscrire pour retrouver les privilèges d'utilisateur enregistré sur le Laboratoire de Biodiversité des Nations Unies.

6. Après avoir enregistré vos modifications, cliquez sur 'RETURN TO MAP VIEW'.

	![](images/en/image019.png)
