# ¿Cómo personalizo las vistas de conjuntos de datos?

Al seleccionar múltiples conjuntos de datos, puede personalizar el mapa ajustando su orden de superposición y opacidad.

1. Para cambiar el orden de superposición, haga clic y mantenga presionado el icono ![](images/icons/order-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"} a la izquierda del nombre del conjunto de datos en la leyenda y mueva el conjunto de datos hacia arriba o hacia abajo según el orden de superposición preferido. El conjunto de datos superior en la leyenda será el conjunto de datos superior en el mapa.

2. Para cambiar la opacidad, haga clic en el icono ![](images/icons/opacity-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"}. Reducir la opacidad aumenta la transparencia del conjunto de datos. Por ejemplo, para visualizar tanto la pérdida de cobertura arbórea como las áreas protegidas, puede posicionar el conjunto de datos de pérdida de cobertura arbórea encima del conjunto de datos de áreas protegidas y ajustar la opacidad de las áreas protegidas al 60%. Esto crea un mapa que muestra la pérdida de cobertura arbórea dentro de las áreas protegidas, así como la pérdida general en todo el país.

3. Para ocultar temporalmente un conjunto de datos en el mapa, haga clic en el icono ![](images/icons/show-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"}. Para hacerlo visible nuevamente, haga clic en el icono ![](images/icons/hide-icon.png){style="display: inline; width: 1em; height: 2em; width: 2em;"}.
