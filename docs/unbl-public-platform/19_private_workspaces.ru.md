# Что такое рабочие пространства UNBL? Как запросить рабочее пространство UNBL?

Рабочие пространства UNBL предоставляют безопасную рабочую область, где данные любого масштаба могут быть добавлены и переданы определенной группе пользователей. Они предлагают безопасную область для совместной работы над вашими проектами и могут обеспечить способ для экспертов по ГИС сотрудничать с нетехническими экспертами, используя удобный интерфейс.

**Три функциональные возможности доступны через рабочие пространства, которые недоступны на публичной платформе**:

1. Возможность загружать места;

2. Возможность добавлять собственные наборы данных и делиться ими приватно и/или публично;

3. Возможность добавить наш инструмент комплексного пространственного планирования ELSA для проведения анализа пространственной приоритизации для вашей страны.

Рабочие пространства UNBL доступны для всех некоммерческих пользователей. Чтобы запросить рабочее пространство, пожалуйста, перейдите на [страницу рабочих пространств Лаборатории биоразнообразия ООН](https://unbiodiversitylab.org/en/unbl-workspaces/) и заполните форму.

Вы можете получить доступ к нашему руководству пользователя по рабочим пространствам UNBL [здесь](../unbl-workspaces/index.md).
