# Что такое рабочие пространство на UNBL? Как мне запросить рабочее пространство UNBL?

Рабочие пространства UNBL предоставляют безопасную рабочую зону, где можно добавлять данные любого объема и делиться ими с группой определенных пользователей. Они предлагают безопасную зону для совместной работы над вашими проектами и могут предоставить возможность экспертам в области ГИС сотрудничать с нетехническими экспертами с помощью удобного интерфейса. 

**Рабочие пространства предоставляют три функции, недоступные на общедоступной платформе:**

1. возможность загружать места;

2. возможность добавлять собственные наборы данных и делиться ими в частном и/или общедоступном режиме;

3. возможность добавлять наш инструмент интегрированного пространственного планирования ELSA для проведения анализа пространственной приоритезации для вашей страны.

Рабочие пространства UNBL доступны для всех некоммерческих пользователей. Чтобы запросить рабочее пространство, перейдите на [страницу рабочих пространств UNBL](https://unbiodiversitylab.org/ru/unbl-workspaces/) и заполните форму.

Вы можете ознакомиться с нашим руководством пользователя по рабочим пространствам UNBL [здесь](../unbl-workspaces/index.md).
