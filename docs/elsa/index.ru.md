# Руководство пользователя Инструмента интегрированного пространственного планирования ELSA на Лаборатории ООН по биоразнообразию

Данное руководство пользователя призвано предоставить четкий и краткий обзор использования Инструмента интегрированного пространственного планирования основных зон обеспечения жизнедеятельности (ELSA) на Лаборатории биоразнообразия ООН (UNBL). Этот инструмент был разработан для оказания поддержки странам в использовании интегрированного пространственного планирования для поддержки реализации Куньмин-Монреальской глобальной рамочной программы по биоразнообразию с помощью подхода системного планирования охраны природы (SCP), поддерживаемого UNBL.

* [Глава 1](01_overview.md) предоставляет обзор и ключевые дополнительные ресурсы по процессу картирования ELSA.
* [Глава 2](02_tool_purpose.md) описывает ключевые функции и применения инструмента ELSA.
* [Главы 3](03_registration.md) - [11](11_support.md) предоставляют пошаговые инструкции по использованию инструмента ELSA на сайте UNBL.
