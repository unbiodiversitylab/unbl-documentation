# Цитирование карт

Если вы хотите использовать окончательные карты, полученные с помощью этого инструмента, в официальном отчете или публикации, пожалуйста, используйте следующую ссылку: 

```
UNDP & UNEP-WCMC 2025. Интегрированное пространственное планирование для ГПБ. Карта создана с использованием пространственных данных и инструмента «ELSA» Лаборатории ООН по биоразнообразию. (https://map.unbiodiversitylab.org/earth) [вставьте дату с указанием дня, месяца и года]. 
```
