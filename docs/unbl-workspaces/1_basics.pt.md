# Noções básicas dos espaços de trabalho UNBL

## O que é um espaço de trabalho UNBL?

Os espaços de trabalho UNBL fornecem uma área de trabalho segura dentro do UNBL onde dados de qualquer escala podem ser adicionados e partilhados com um conjunto de utilizadores especificados. Oferecem um espaço seguro para colaborar nos seus projetos e podem proporcionar uma forma de os especialistas em SIG colaborarem com especialistas não técnicos usando uma interface amigável.

**Três funcionalidades estão disponíveis através dos espaços de trabalho que não estão disponíveis na plataforma pública:** (1) capacidade de carregar lugares e exibir métricas dinâmicas para eles; (2) capacidade de adicionar os seus próprios conjuntos de dados e partilhá-los de forma privada e/ou pública; (3) capacidade de aceder à ferramenta de planeamento espacial integrado ELSA para executar uma análise de priorização espacial para o seu país. Enquanto as funcionalidades (1) e (2) estão disponíveis por defeito, a ferramenta de análise ELSA precisa de ser solicitada.

## Como solicito um espaço de trabalho UNBL?

Os espaços de trabalho UNBL estão disponíveis para todos os utilizadores não comerciais. Para solicitar um espaço de trabalho, por favor navegue até à [página de espaços de trabalho do UN Biodiversity Lab](https://unbiodiversitylab.org/en/unbl-workspaces/) e preencha o formulário. Se desejar acesso à ferramenta ELSA, por favor indique-o no formulário. Para saber mais sobre ELSA, por favor visite a página de apresentação [Mapeamento de Áreas Essenciais de Suporte à Vida (ELSA)](https://unbiodiversitylab.org/en/integrated-spatial-planning/mapping-essential-life-support-areas-elsas/).
