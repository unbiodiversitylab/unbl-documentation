# Навигация по интерфейсу администрирования рабочего пространства

## Как получить доступ к интерфейсу администрирования?

Для добавления и управления пользователями, местами и наборами данных в вашем рабочем пространстве вам необходимо получить доступ к интерфейсу администрирования вашего рабочего пространства. Для этого:

1.	Нажмите на кнопку «WORKSPACES» в левом верхнем углу.

2.	Выберите кнопку «ADMIN», связанную с рабочим пространством по вашему выбору.

3.	Страница администрирования вашего рабочего пространства также доступна по следующему URL:

>https://map.unbiodiversitylab.org/admin/[SLUG-ИМЯ-ВАШЕГО-РАБОЧЕГО-ПРОСТРАНСТВА]

## Какие компоненты доступны в интерфейсе администрирования?

Вы можете перемещаться по интерфейсу администрирования с помощью выпадающего меню в верхней части левой панели. В зависимости от вашей роли в рабочем пространстве вы можете управлять *Users*, *Places*, *Layers*, *Widgets* и *Dashboards*.

!!!Note
	Функции Widget и Dashboard находятся в разработке и в настоящее время недоступны.

Для доступа к различным компонентам:

1.	Нажмите на кнопку «Home», чтобы раскрыть выпадающее меню.

2.	Выберите компонент, который вы хотите просмотреть. Дополнительная информация о каждом компоненте предоставлена в последующих разделах данного руководства пользователя.
