# Navigating the Workspace Admin Interface

## How do I access the admin interface?

To add and manage users, places, and datasets in your workspace, you need to access the admin interface of your workspace. To do this:

1.	Click on the ‘WORKSPACES’ button in the top-left corner.
  
2.	Select the ’ADMIN’ button associated with the workspace of your choice.

3.	Your workspace admin page can also be accessed at the following URL:  

>https://map.unbiodiversitylab.org/admin/[YOURWORKSPACESLUGNAME]

![](images/en/image7.png)

## What components are available within the admin interface?

You can navigate the admin interface by using the dropdown menu in the upper section of the left-hand panel. Depending on your workspace role you can manage *Users*, *Places*, *Layers*, *Widgets* and *Dashboards*. 

!!!Note
	Widget and Dashboard features are in development and unavailable at this time.  

To access the various components:

1.	Click on the ‘Home’ button to expand the dropdown menu.

2.	Select the component you would like to view. More information on each component is provided in later sections of this user guide.
